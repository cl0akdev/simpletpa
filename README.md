# simpletpa

### SimpleTpa for luanti (formerly minetest)
### this is basically a port lol
### also this account is brandnew during this time just to upload this mod I am not an alt or smth

Features
- /tpa <player> — request to teleport to a player
- /tpaccept — accept the last request
- /tpdeny — deny the last request

Usage
- /tpa <player>
- /tpaccept
- /tpdeny

Configuration
- Edit `init.lua` to change request timeout (EXPIRE) or add cooldowns/permissions.
